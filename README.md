# WebApp—H5小说阅读器功能

## 使用技术：

HTML5/CSS3/JavaScript/zepto.js/jsonp/base64

## 实现功能：

1.点击内容区域唤醒字体设置面板，滑动页面 面板隐藏。

2.点击上一章/下一章更换内容并跳转至页面顶部。

3.字体大小切换(最大20px，最小12px)。

4.多种背景颜色切换。

5.白天/夜间模式切换。

6.可存储用户上次设置参数。
